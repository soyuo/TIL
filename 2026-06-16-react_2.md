# [React](https://www.npmjs.com/package/react)#2
 - page 나누기

> [!NOTE]
> 해당 학습에선 `react-router-dom` 을 추가로 사용함
> 
> 또한, TypeScript 로 시작함

## introduction
만약, 기본으로 지원되는 `/` 이 아닌 `/1`이나 `/2` 같이
페이지를 나눠서 두고 싶다면 어떻게 해야 할까?

이를 위해선 라우팅을 설정하면 된다.

```cmd
npm i react-router-dom
```

라우팅을 위해 `react-router-dom` 을 설치한다.

## struct
```
src/
├── pages/
│   ├── HelloPage.tsx   ← rendering at /1
│   └── WorldPage.tsx   ← rendering at /2
├── App.tsx             ← router setting
└── main.tsx
```

> ### main.tsx
```tsx
import React from 'react';
import ReactDOM from 'react-dom/client';
import App from './App';

ReactDOM.createRoot(
    document.getElementById('root')!
).render(
    <React.StrictMode>
        <App />
    </React.StrictMode>
);
```

> ### App.tsx
```tsx
import { BrowserRouter, Routes, Route } from 'react-router-dom';
import HelloPage from './pages/HelloPage';
import WorldPage from './pages/WorldPage';

export default function App() {
    return (
        <BrowserRouter>
            <Routes>
                <Route path="/1" element={<HelloPage />} />
                <Route path="/2" element={<WorldPage />} />
            </Routes>
        </BrowserRouter>
    );
}
```

> ### pages/HelloPage.tsx
```tsx
export default function HelloPage() {
    return <h1>Hello</h1>;
}
```

> ### pages/WorldPage.tsx
```tsx
export default function WorldPage() {
    return <h1>World</h1>;
}
```

### 이를 통해, Hello와 World를 나누어 각각 다른 페이지에서 볼 수 있다.
