# 백엔드(Backend)란 무엇인가?

백엔드(Backend)는 우리가 웹사이트나 애플리케이션을 사용할 때 **사용자 눈에는 보이지 않지만 실제로 서비스가 작동하도록 만드는 것**을 의미한다.  
간단히 말해 **데이터를 받고 → 처리하고 → 저장하고 → 다시 전달하는 역할을 담당하는 영역**이다.

우리가 웹사이트에서 회원가입을 하거나, 로그인하고, 게시글을 작성하거나, 상품을 구매할 때 모든 정보는 어딘가에서 처리되고 저장되어야 한다.  
이러한 작업을 담당하는 부분이다.

---

## 프론트엔드와 백엔드의 차이

웹 서비스는 일반적으로 두 영역으로 나뉜다.

| 구분 | 역할 |
|---|---|
| 프론트엔드 (Frontend) | 사용자가 직접 보는 화면 |
| 백엔드 (Backend) | 보이지 않는 곳에서 데이터 처리 및 관리 |

예를 들어 로그인 기능을 봐보자.

1. 사용자가 로그인 버튼을 누른다 (Frontend)
2. 아이디와 비밀번호가 서버로 전송된다
3. 서버가 데이터베이스에서 사용자 정보를 찾는다
4. 비밀번호가 맞는지 검사한다
5. 결과를 다시 사용자에게 전달한다

이 과정에서 **2~5번을 담당하는 부분**이 백엔드다.
