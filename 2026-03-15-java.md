## 기본 문법

### Hello World
```java
public class Hello {
    public static void main(String[] args) {
        System.out.println("Hello World");
    }
}
```
> `class` → 기본 단위 / `main()` → 시작 지점 / `System.out.println()` → 콘솔 출력

### 주석
```java
// 한 줄 주석

/*
  여러 줄 주석
*/
```
> 실행에 영향 없음. 코드 설명 용도.

## 변수와 자료형

### 변수 선언
```
자료형  변수이름  =  값;
```
```java
int age = 20;
String name = "Tom";
```

### 기본 자료형
| 타입 | 설명 | 예시 |
|------|------|------|
| `int` | 정수 | `int age = 20;` |
| `double` | 실수 | `double height = 175.5;` |
| `boolean` | 참/거짓 | `boolean isStudent = true;` |
| `char` | 문자 1개 | `char grade = 'A';` |
| `String` | 문자열 | `String name = "Kim";` |

### 변수 명명 규칙
- X 숫자로 시작 불가
- X 공백 사용 불가
- X 예약어 사용 불가 (`int`, `class` 등)
- O **camelCase** 사용 권장 → `studentAge`, `totalScore`

## 연산자

### 산술 연산자
```java
int a = 10 + 5;   // +, -, *, /
```

### 문자열 연결
```java
String name = "Tom";
System.out.println("Hello " + name);  // Hello Tom
```

### 증감 연산자
| 연산자 | 동작 |
|--------|------|
| `++a` / `--a` | 먼저 증감, 그 다음 사용 |
| `a++` / `a--` | 먼저 사용, 그 다음 증감 |

### 비교 연산자
`>` / `>=` / `<` / `<=` / `==` / `!=`

### 논리 연산자
```java
if (age > 18 && age < 30) { ... }  // && AND
if (a > 0 || b > 0) { ... }        // || OR
if (!isStudent) { ... }             // !  NOT
```

## 조건문

### if / else if / else
```java
int score = 85;

if (score >= 90) {
    System.out.println("A");
} else if (score >= 80) {
    System.out.println("B");
} else {
    System.out.println("C");
}
```

### switch 문
```java
// 기존 방식
switch(num) {
    case 1:
        System.out.println("One");
        break;
    case 2:
        System.out.println("Two");
        break;
    default:
        System.out.println("Other");
}

// Java 14+ 방식
switch(num) {
    case 1 -> System.out.println("One");
    case 2 -> System.out.println("Two");
    default -> System.out.println("Other");
}
```

### 삼항 연산자
```java
String result = (age >= 20) ? "성인" : "미성년자";
```
> 간단한 조건을 한 줄로 처리

## 반복문

### while — 조건이 true인 동안 반복
```java
int i = 0;
while (i < 5) {
    System.out.println(i);
    i++;
}
```

### do-while — 최소 1번은 무조건 실행
```java
int i = 0;
do {
    System.out.println(i);
    i++;
} while (i < 5);
```

### for — 반복 횟수가 정해진 경우
```java
for (int i = 0; i < 5; i++) {
    System.out.println(i);
}
// 구조: (초기값 → 조건 → 증감)
```

### 중첩 반복문
```java
for (int i = 0; i < 3; i++) {
    for (int j = 0; j < 3; j++) {
        System.out.println(i + " " + j);
    }
}
```

### break / continue
```java
// break → 반복문 즉시 종료
if (i == 5) break;

// continue → 해당 반복만 건너뜀
if (i == 5) continue;
```

## 스코프 (Scope)

> 변수가 **사용 가능한 범위**를 의미한다.

```java
public static void main(String[] args) {
    int a = 10;             // main 전체에서 사용 가능

    if (true) {
        int b = 20;         // if 블록 안에서만 사용 가능
        System.out.println(a);
        System.out.println(b);
    }

    System.out.println(b);  // X
}
```

## 형변환 (Type Casting)

### 자동 형변환 — 작은 타입 → 큰 타입
```java
int a = 10;
double b = a;   // 자동 변환
```

### 명시적 형변환 — 큰 타입 → 작은 타입
```java
double a = 10.5;
int b = (int) a;    // 소수점 잘림 → 10
```

### 연산 중 형변환
```java
int a = 10;
double b = 3;
double result = a / b;   // 정수 ÷ 실수 → 실수 결과
```
