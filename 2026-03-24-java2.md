# Java Basic

## 입력 (Scanner)

### Scanner 사용

```java
import java.util.Scanner;

Scanner sc = new Scanner(System.in);
```

### 주요 입력 메소드

```java
String str = sc.nextLine();   // 문자열 입력
int num = sc.nextInt();       // 정수 입력
double d = sc.nextDouble();   // 실수 입력
```

### 주의사항

```java
int num = sc.nextInt();
sc.nextLine();

String str = sc.nextLine();
```

> `nextInt()` 후 바로 `nextLine()` 쓰면 개행이 남아있어서 문제가 발생할 수 있음

## 배열 (Array)

### 배열 선언 및 생성

```java
int[] arr = new int[5];
```

### 배열 초기화

```java
int[] arr = {1, 2, 3, 4, 5};
```

### 배열 접근

```java
arr[0] = 10;           // 값 변경
System.out.println(arr[0]);  // 값 출력
```

### 인덱스 범위

```
0 ~ (length - 1)
```

### 배열 길이

```java
arr.length
```

## 배열 리팩토링 (코드 개선)

### 잘못된 방식

```java
int a = 10;
int b = 20;
int c = 30;
```

### 개선된 방식

```java
int[] arr = {10, 20, 30};

for (int i = 0; i < arr.length; i++) {
    System.out.println(arr[i]);
}
```

> 여러 데이터를 다룰 때는 **배열 사용**

## 2차원 배열

### 선언

```java
int[][] arr = new int[3][3];
```

### 초기화

```java
int[][] arr = {
    {1, 2, 3},
    {4, 5, 6},
    {7, 8, 9}
};
```

### 접근

```java
System.out.println(arr[0][1]);  // 2
```

### 반복문 사용

```java
for (int i = 0; i < arr.length; i++) {
    for (int j = 0; j < arr[i].length; j++) {
        System.out.print(arr[i][j] + " ");
    }
    System.out.println();
}
```

---

## 2차원 배열 리팩토링

### 개선 포인트

* 중첩 반복문 구조 활용
* 행(row) / 열(column) 개념 이해

```java
for (int[] row : arr) {
    for (int value : row) {
        System.out.print(value + " ");
    }
    System.out.println();
}
```

> 코드 가독성이 훨씬 좋아짐

## 향상된 for문 (Enhanced For Loop)

### 기본 구조

```java
for (int number : numbers) {
    System.out.println(number);
}
```

### 특징

* 배열 / 컬렉션 순회에 사용
* 인덱스 접근 불가
* 읽기 전용에 적합

## 메소드 (Method)

> 특정 기능을 수행하는 코드 블록 (함수)

### 기본 구조

```java
public static returnType methodName(parameter) {
    // 실행 코드
    return 값;
}
```

### 예제

```java
public static int add(int a, int b) {
    return a + b;
}
```

### void 메소드

```java
public static void printHello() {
    System.out.println("Hello");
}
```

---

## 메소드 구성 요소

| 요소          | 설명                           |
| ----------- | ---------------------------- |
| Modifier    | 접근 제한자 (`public`, `private`) |
| Return Type | 반환 타입 (`int`, `void`)        |
| Method Name | 메소드 이름                       |
| Parameter   | 입력 값                         |
| Method Body | 실행 코드                        |

---

## 중요 규칙

### 반환 타입

```java
public static int test() {
    return 10;   // OK
}
```

```java
public static int test() {
    return "hello";   // X (타입 불일치)
}
```

> **반환 값의 타입과 반환 타입은 반드시 같아야 한다**

---

### 매개변수 & 인자

```java
public static void print(int num) { }

print(10);     // OK
print("hi");   // X
```

> **매개변수 타입 = 인자 타입 반드시 일치**

## 메소드 오버로딩 (Overloading)

> 같은 이름의 메소드를 여러 개 정의하는 것

### 가능 조건

* 메소드 이름 동일
* 매개변수 **개수 or 타입 or 순서** 달라야 함

### 예제

```java
public static int add(int a, int b) {
    return a + b;
}

public static double add(double a, double b) {
    return a + b;
}

public static int add(int a, int b, int c) {
    return a + b + c;
}
```

### 불가능한 경우

```java
public static int add(int a, int b) { }
public static double add(int a, int b) { }  // X
```

> **반환 타입만 다른 것은 오버로딩 불가능**
