# [React](https://www.npmjs.com/package/react)#1
 - 메타에서 개발한 프론트엔드 프레임워크
 - 오픈 소스 Javascirpt 라이브러리

> [!NOTE]
> 해당 학습에선 vite 로 react 를 시작함
>
> 또한, TypeScript 로 시작함

## init
```cmd
npm create vite@latest first-app -- --template react-ts
y
cd first-app
npm install
```

## start
```cmd
npm run dev
```

## struct
```
src/
├─ App.tsx
├─ main.tsx
```

> ### main.tsx
```tsx
import React from 'react';
import ReactDOM from 'react-dom/client';
import App from './App';

ReactDOM.createRoot(
    document.getElementById('root')!
).render(
    <React.StrictMode>
        <App />
    </React.StrictMode>
);
```

> ### App.tsx
```tsx
export default function App() {
    return (
        <h1>Hello world!</h1>
    );
}
```

## component
> ### Hello.tsx
```tsx
function Hello() { // Not 'h'ello, must be 'H'ello!
    return (
        <h1>Hello, everyone</h1>
    );
}

export default Hello;
```

> ### Usage
```tsx
import Hello from './Hello';

export default function App() {
    return (
        <Hello />
    );
}
```

**Result** : `Hello, everyone`
