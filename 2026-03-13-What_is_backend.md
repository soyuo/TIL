# 백엔드(Backend)란?

백엔드(Backend)는 우리가 웹사이트나 애플리케이션을 사용할 때 **사용자 눈에는 보이지 않지만 실제로 서비스가 작동하도록 만드는 것**을 의미한다.  
#### 데이터를 받고 → 처리하고 → 저장하고 → 다시 전달하는 역할을 담당하는 영역

우리가 웹사이트에서의 모든 정보는 어딘가에서 처리되고 저장되어야 한다.  
이러한 작업을 담당하는 부분이다.

---

## Frontend vs Backend

웹 서비스는 일반적으로 두 영역으로 나뉜다.

| 구분 | 역할 |
|---|---|
| 프론트엔드 (Frontend) | 사용자가 직접 보는 화면 |
| 백엔드 (Backend) | 보이지 않는 곳에서 데이터 처리 및 관리 |

---

### Example Process

1. 로그인 버튼 인식 (Frontend)
2. 아이디와 비밀번호가 서버로 전송
3. 서버가 데이터베이스에서 사용자 정보 검색
4. 비밀번호 검증
5. 결과를 사용자에게 재전달

Backend 는 **2~5번을 담당하는 부분**
